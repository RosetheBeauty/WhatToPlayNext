# What To Play Next: Gaming Discovery Hub

Welcome to **What To Play Next**, a gaming-focused website designed to help users discover, explore, and track video games they are interested in. This project gives users an easy way to find new games, view details, and decide what to play next without feeling overwhelmed by too many options.

---

## About the Project

This website project was created as part of my class assignment. The goal is to build a simple and organized gaming hub where users can browse different video games and quickly learn basic information about them.

The site focuses on making game discovery easier by showing clear game categories, short descriptions, and an easy browsing experience. It is designed to be simple, fast, and easy to understand for all users.

---

## Key Features of the Website

- **Game List Page:** Displays a collection of video games with images, titles, and short descriptions.  
- **Search Feature:** Allows users to search for games by name.  
- **Category Filtering:** Lets users filter games based on genre like action, RPG, or multiplayer.  
- **Game Details View:** Shows more information when a user clicks on a game.  
- **Favorites Feature (planned):** Lets users save games they want to play later.  
- **Clean Layout:** Simple design that makes browsing easy and clear.  
- **Responsive Design (planned):** Will work on both desktop and mobile devices.

---

## Repository Structure

The repository contains:

- **index.html** – The homepage showing the game list and main layout.  
- **style.css** – The styling for layout, colors, and design.  
- **script.js** – Adds interactivity like search and filtering.  
- **images/** – Folder containing game images and visual assets.  
- **README.md** – Project documentation describing the website.
